Getting Started

Sandbox is a native desktop application. Double-click the Sandbox shortcut or executable to launch it. The app opens in its own window — no browser or terminal needed.

Your settings, tags, favorites, and thumbnails are stored locally and persist between sessions.

Home Page

The home page displays all available miniapps as cards or list items. Use the sidebar to filter by tag, the search bar to find apps by name, and the sort dropdown to change the display order.

Miniapps

Each miniapp is a self-contained tool accessible from the home page. Click any app card to open it. Use the Home button or Close button in the miniapp's top bar to return to the home page.

Tags

Tags categorize your miniapps. Click a tag in the sidebar to filter the home page to only show apps with that tag. Tags are displayed in alphabetical order.

Creating Tags

Click + Add Tag in the sidebar to create a new tag. Enter a name and press Enter or click Add.

Deleting Tags

Hover over a tag in the sidebar and click the X button that appears. You will be asked to confirm. Deleting a tag removes it from all apps that had it assigned.

Assigning Tags to Apps

Open the edit modal for an app (pencil icon) and use the Tags dropdown to check/uncheck tags. Tags in the dropdown are listed alphabetically.

Favorites

Click the star icon on any app card to mark it as a favorite. Favorited apps appear in a dedicated section at the top of the home page when viewing All Apps in Custom Order.

Search and Sort

Use the search bar in the top bar to filter apps by name or description. The sort dropdown lets you order apps by custom order, alphabetical, recently used, or date added.

Card and List View

Toggle between card view (grid of cards with thumbnails) and list view (compact rows) using the view toggle buttons in the top bar. Your preference is saved locally.

Editing App Details

Hover over an app card and click the edit (pencil) icon to open the edit modal. From here you can change the app's title, description, assigned tags, and thumbnail image.

Thumbnails

Each app can have a custom thumbnail image displayed on its card. In the edit modal, click Choose Image... to open a file picker and select an image file (PNG, JPG, GIF, or WebP).

FW Allocation

Allocates field weld counts from a pivot table to BW/SW labor rows in an Excel file. Prioritizes the largest pipe size first (header weld), then distributes by longest pipe run footage.

Usage

  1. Click Choose File... to select the FW Count file (Excel with Drawing Number and weld count columns)
  2. Click Choose File... to select the Labor file (Excel with Drawing Number, Component, Size, Quantity, and ShopField columns)
  3. Click Run Allocation to process
  4. Review the results summary (welds flipped, drawings processed, any issues)
  5. Click Download Updated Labor to save the modified file
  6. Click Download Issues Log to save any issues encountered
